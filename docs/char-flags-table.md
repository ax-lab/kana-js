Character classification Table
==============================

References:
- https://en.wiktionary.org/wiki

Flag descriptions:

- HIRAGANA    — Any character part of a Hiragana word element.
- KATAKANA    — Any character part of a Katakana word element.
- KANJI       — Any character in the kanji set, including non Japanese.
- ROMAN       — Full-width versions of ASCII letters and digits.
- ROMAJI      — Part of the set of characters generated by Romaji
                transliteration of single words, including symbols.

- SOUNDMARK   — Prolonged sound mark.
- SMALL       — Small version of a Kana character.
- MARK        — In-word non-letter character that affects transliteration.

- RARE        — Rare or archaic Kana characters.
- COMBINING   — Combining Unicode character.
- FULLWIDTH   — Fullwidth Japanese characters that represent ASCII characters.
- HALFWIDTH   — Halfwidth versions of Japanese characters.

- DIGIT       — Decimal digit.
- LETTER      — Kana or romaji letter.
- SPACE       — Whitespace characters. Note: not considered as PUNCTUATION.
- PUNCTUATION — Symbol that serves as a word/sentence separator. Moreover, those
                characters have a transliteration counterpart in ASCII/Japanese.
- SYMBOL      — Generic symbol. This is the general classification for a non-word
                graphical character with no particular grammatical significance.

- JAPANESE    — Any Japanese characters. This allows differentiation for flags
                that apply to both ASCII and Japanese characters.
- ASCII       — ASCII characters.

Individual character listing:

- Characters are divided by their general category.
- Each line is tab-separated.
- Columns are: `UNICODE`, `CHAR`, `FLAGS`, `UNICODE NAME`
- Common flags are listed at the beginning and omitted for all characters.
- If a character only contains the common flags, its `FLAGS` will be `*`.

Observations:

- `ゎ` — Small combining form of hiragana わ (wa) used in the archaic 拗音 (yōon)
  morae くゎ (kwa) and ぐゎ (gwa) — e.g. "くゎんおん (Kwan'on)".
- The hiragana `ゔ`, `ゕ` and `ゖ` were introduced for hiragana-katakana
  correspondence with `ヴ`, `ヵ` and `ヶ` in the Japanese input method. The
  alternative forms have no meaningful usage.
- `ゟ` — Ligature of よ (yo) and り (ri). Dated abbreviation of より.
- `ゝ` — Dated hiragana iteration mark, indicating that the previous hiragana
  character should be repeated. E.g. "さゝき (Sasaki)".
- `ゞ` — Dated hiragana voiced iteration mark, indicating that the previous hiragana
  character should be repeated with a voice mark applied. E.g. "あひゞき (Ahibiki)".

Observations:

- `ヵ` — `(-ka)` — Generic counter. E.g. "一ヵ月 (ikkagetsu)"
- `ヶ`
	- `ヶ or か • (-ka)` — An abbreviation for the kanji 箇 (ka), the counter
	  for months, places or provisions. Note that ヶ is a graphical abbreviation
	  of the top of the character (⺮, 竹), and is pronounced カ ka (or ガ ga),
	  not as ケ ke. E.g. "三ヶ月 (sankagetsu) / 三ヶ国 (sankakoku)"
	- `ヶ or こ • (-ko)` — 個: General counter for inanimates (e.g. "３ヶ sanko").
	- `ヶ or が • (ga)` — An alternative spelling of an obsolete possessive
	  marker が. Found only in place names. E.g. "藤ヶ丘 (Fujigaoka)"
- `ヷ`, `ヸ`, `ヹ`, `ヺ` — The katakana syllables for va, vi, ve, and vo. Their
  equivalent in hiragana are the composites `わ゙`, `ゐ゙`, `ゑ゙`, and `を゙`. Those
  kana are rarely used, with `ヴァ`, `ヴィ`, `ヴェ`, and `ヴォ` seeing more usage.
- `ヽ` — Dated katakana iteration mark, indicating that the previous katakana
  character should be repeated. E.g. "ホトヽギス (hototogisu)"
- `ヾ` — Dated katakana voiced iteration mark, indicating that the previous
  katakana character should be repeated with a voice mark (dakuten) applied.
  E.g. "サヾエ (Sazae)".
- `ヿ` — Ligature of コ (ko) and ト (to). Archaic abbreviation of 事 (koto).
- Ranges `U+31F0` to `U+31FF` are part of the phonetic extensions for Ainu.

Common Flags: HIRAGANA | KATAKANA | JAPANESE

```
U+3099	◌゙	COMBINING            	Combining Katakana-Hiragana Voiced Sound Mark
U+309A	◌゚	COMBINING            	Combining Katakana-Hiragana Semi-Voiced Sound Mark
U+30FC	ー	SOUNDMARK            	Katakana-Hiragana Prolonged Sound Mark
U+303C	〼	MARK                 	Masu Mark
U+FF70	ｰ	SOUNDMARK | HALFWIDTH	Halfwidth Katakana-Hiragana Prolonged Sound Mark
```

Observation:

- The combining marks are flagged as Kana so that they will be included as part
  of a kana group.
- `〼` (Masu Mark) — Used pictographically as an image of a traditional masu
  square measuring cup. Used phonetically as shorthand for the masu polite verb
  ending, by homophony of 枡 (masu) and the suffix ます (masu). Popular shorthand
  in the Edo period. The use of this symbol for the verb suffix ます (masu)
  became less common during the 20th century. Now with the advent of IME
  software and texting on mobile phones, use of this symbol is increasing once
  again. It is now garnering increasing application in replacing ます elsewhere
  as well, such as in ますます (masumasu).
