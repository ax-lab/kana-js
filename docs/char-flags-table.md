Character classification Table
==============================

References:
- https://en.wiktionary.org/wiki

Flag descriptions:

- HIRAGANA    ‚Äî Any character part of a Hiragana word element.
- KATAKANA    ‚Äî Any character part of a Katakana word element.
- KANJI       ‚Äî Any character in the kanji set, including non Japanese.
- ROMAN       ‚Äî Full-width versions of ASCII letters and digits.
- ROMAJI      ‚Äî Part of the set of characters generated by Romaji
                transliteration of single words, including symbols.

- SOUNDMARK   ‚Äî Prolonged sound mark.
- SMALL       ‚Äî Small version of a Kana character.
- MARK        ‚Äî In-word non-letter character that affects transliteration.

- RARE        ‚Äî Rare or archaic Kana characters.
- COMBINING   ‚Äî Combining Unicode character.
- FULLWIDTH   ‚Äî Fullwidth Japanese characters that represent ASCII characters.
- HALFWIDTH   ‚Äî Halfwidth versions of Japanese characters.

- DIGIT       ‚Äî Decimal digit.
- LETTER      ‚Äî Kana or romaji letter.
- SPACE       ‚Äî Whitespace characters. Note: not considered as PUNCTUATION.
- PUNCTUATION ‚Äî Symbol that serves as a word/sentence separator. Moreover, those
                characters have a transliteration counterpart in ASCII/Japanese.
- SYMBOL      ‚Äî Generic symbol. This is the general classification for a non-word
                graphical character with no particular grammatical significance.

- JAPANESE    ‚Äî Any Japanese characters. This allows differentiation for flags
                that apply to both ASCII and Japanese characters.
- ASCII       ‚Äî ASCII characters.

Individual character listing:

- Characters are divided by their general category.
- Each line is tab-separated.
- Columns are: `UNICODE`, `CHAR`, `FLAGS`, `UNICODE NAME`
- Common flags are listed at the beginning and omitted for all characters.
- If a character only contains the common flags, its `FLAGS` will be `*`.
